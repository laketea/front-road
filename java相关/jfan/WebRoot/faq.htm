常见问题

  • 如何使用现金券？

    当您获得现金券之后，在下订单的时候就可以在页面右侧的输入框中输入现金券号码即可减免付款数额。一个订单只能使用一张现金券。现金券是有期限限 制，请您在有效期内使用哦。

  • 我在聚美优品问答中提问了，为什么没有回答，或者回答很慢呢？

    非常抱歉！我们每天服务数以十万计的用户，每一名客服同时要面对数十位客户的提问，因此如果有回复较慢的情况出现， 敬请您谅解！我们会查看并且回答每一位用户的问题。如果您想第一时间获得解答，建议您拨打聚美优品的客服热线 4000-123-888 咨询。

  • 为什么支付宝是直接到帐付款不是担保交易呢？

    支付宝的担保交易是针对淘宝等C2C网站专门设计的，聚美优品这样的大型B2C网站拥有很高的资质，使用的在线支付方式为支付宝即时到帐交易。我们是支付宝 特约商户，已经加入支付宝信任计划，请大家放心消费。若在团购结束之后您的订单尚未付款，会自动过期。

  • 我已经购买了，什么时候发货？

    如果您选择“尽快发货”，正常情况下您的订单将在1个工作日内发出。

    如果您选择“推迟发货”，正常情况下您的订单将在您指定的发货日期发出。

    遇有订单高峰或者特殊物流情况，您的订单发货可能会稍有延迟，请谅解。

    在我们发货之后，您将收到邮件和短信告知包裹的快递单号（发给您在购买时填写的手机号码），快递公司可能信息更新有一段时间的延迟，请您在第二天查询 快递动态。如有个别快递进度不更新的情况，可能是由于您的订单被快递公司用EMS转运导致，请联系我们的客服查询。

  • 如何付款？

    聚美优品目前支持用支付宝、财付通、快钱等国内领先的第三方支付平台，支持国内数十家银行的网上支付，确保您的交易安全。

  • 邮费怎么计算？

    聚美优品默认使用申通快递，邮费五元。支持圆通和中通，邮费也是五元，还可以选择EMS发货，邮费15元。

    聚美优品同时为您提供了多种免邮费购买的优惠政策，请点击这里查看。

  • 为什么我收到了发货通知的短信，但是去快递公司查询快递单号显示没有信息？

    快递单号从快递公司扫描到录入系统需要一段时间处理，请大家不要着急。一般来说您在收到发货通知的第二天就肯定能查看到物流信息了。

  • 团购成功后，我还能购买么？

    团购成功后，仍可以继续购买。但是请注意个别团购有数量上限，先买先得，卖完为止。

  • 我付款成功了，但后台显示团购已结束，这是什么情况？

    当付款与团购结束同步进行时会发生上述情况，您支付成功后的款项将直接打入您的聚美优品帐户，如有疑问可致电4000-123-888。

  • 我收到了聚美优品礼品卡，如何领取？

    请在登录后进入“我的余额”页面，在右侧的小工具中输入您获得的礼品卡号码，按充值按钮即可。

  • 我对商品不满意，可以进行退货吗？

    当然可以。在聚美优品，您理应享受到最顶级的服务。退货自然也不例外。

    聚美优品目前提供提出化妆品零售业最高售后标准――30天拆封无条件退货。所有从聚美优品购买的商品，自收 货之日起，30天内可无任何条件退回货物并获得全额退款――即使您已开封甚至已使用！退货运费也完全由聚美承担（退回货品的全款和寄回的费用，不包含聚美优品邮寄给您的运费）
